LogicScript

LogicScript is a language made specifically for Logic World whose job is to define the behaviour of a component with inputs and outputs. For reference it's best to visit its own wiki, this page only contains a small primer.

LogicScript files (or LSX files) are plain text files, and each defines the behaviour of a single component. Keep in mind that a single script can be reused for multiple components. You can use any text editor to create and modify LSX files like Sublime Text.

Cases

LSX files are divided in sections called cases, each of which contains statement that get executed whenever the component gets an update.

when in[0] == 1
    # Do stuff
end

once
    # Do stuff
end

any
    # Do stuff
end

There are three kinds of cases: when cases, any cases and once cases. any cases are run every time the component gets an update, regardless of the component's inputs. when cases are like any cases, except they only run when the condition is evaluated to a non-zero value. once cases only run when the component is first placed.

Statements

A single statement can for example set an output's value, or conditionally execute some other instructions.

any
    out = 1010
    out = 123'

    if in[1] == 0
        out[1] = 1
    end
end
